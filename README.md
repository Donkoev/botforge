# BotForge

BotForge - это панель управления для создания и менеджмента Telegram ботов-зеркал. Позволяет управлять ботами, пользователями, создавать рассылки и просматривать статистику.

## Стек технологий

-   **Backend**: Python, FastAPI, SQLAlchemy (Async), Aiogram 3, Pydantic, PostgreSQL, Redis.
-   **Frontend**: React, TypeScript, Vite, Ant Design, Recharts.
-   **Infrastructure**: Docker, Docker Compose, Nginx.

## Требования

-   Docker и Docker Compose

## Быстрая установка (One-line)

Для быстрой установки на чистый сервер (Ubuntu/Debian/CentOS) выполните команду:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Donkoev/botforge/main/install.sh)
```

Скрипт автоматически:
1.  Установит Docker и Git (если нет).
2.  Склонирует репозиторий.
3.  Предложит настроить пароли и доступы.
4.  Запустит проект.

## Установка и запуск (Вручную)

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/botforge.git
    cd botforge
    ```

2.  **Настройте переменные окружения:**
    Создайте файл `.env`. Можно скопировать из примера:
    ```bash
    cp .env.example .env
    ```
    Отредактируйте `.env`, указав свои настройки (БД, секреты, логин/пароль админа).

3.  **Запуск в Production режиме:**
    ```bash
    docker-compose up -d --build
    ```
    Панель будет доступна по адресу: `http://localhost`

4.  **Запуск в Development режиме (с hot-reload):**
    ```bash
    docker-compose -f docker-compose.dev.yml up --build
    ```
    -   Frontend: `http://localhost:5173`
    -   Backend API: `http://localhost:8000`
    -   Backend Docs: `http://localhost:8000/docs`

## Создание первого пользователя

При первом запуске, если база пустая, вы можете создать пользователя через API или вручную в БД.
В текущей реализации, миграции создают таблицы автоматически.
Для создания админа можно использовать эндпоинт регистрации (если он открыт) или скрипт.

*Примечание: В текущей версии API `/auth/register` скрыт или отсутствует. Для добавления администратора подключитесь к БД и добавьте запись в таблицу `users`.*

## Функционал

-   **Дашборд**: Общая статистика и графики.
-   **Боты**: Добавление ботов по токену, включение/выключение, настройка приветственных сообщений.
-   **Пользователи**: Просмотр списка пользователей, фильтрация.
-   **Рассылки**: Создание массовых рассылок с кнопками и медиа.
-   **Статистика**: Детальные метрики по дням.
